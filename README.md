# Horror Clock 恐怖时钟 👁️

一个具有恐怖氛围的时钟校准程序，模拟系统故障、时空错位等诡异效果。

![screenshots.png](https://github.com/user-attachments/assets/3d95f42b-91d6-4ed3-815f-691e1a9ec570)
**只需要把main.cpp下载下来就行了。用Visual Studio编译运行。**

## ✨ 特性

- 🔵 **蓝屏风格界面** - 模拟Windows严重错误界面
- ⏰ **时钟冻结与错位** - 诡异的时间流动异常效果
- 💾 **DOS扫描过渡** - 复古的命令行扫描动画
- ⚠️ **黑屏红字警告** - 压迫感十足的最终警告

## 🛠️ 编译说明

### 使用 Microsoft Visual Studio

1. 打开 Visual Studio
2. 创建新项目 → 空项目
3. 将 `main.cpp` 添加到项目
4. 按 `Ctrl+F5` 编译运行

**注意**：程序使用了Windows API（`windows.h`），仅支持Windows平台。
